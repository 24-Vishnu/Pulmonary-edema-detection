# Pulmonary Edema Detection from Chest X-Ray Images using Machine Learning

**About the Project**

This project was developed out of my strong interest in applying Machine Learning to solve real-world healthcare challenges. Rather than building a conventional academic project, I wanted to explore how AI can assist in the early detection of critical medical conditions.

The goal of this project is to detect Pulmonary Edema from chest X-ray images using Machine Learning techniques. Through this work, I gained hands-on experience in medical image analysis, data preprocessing, model training, evaluation, and web-based deployment.

This project reflects my curiosity for both Machine Learning and healthcare technology, and it represents my personal effort to learn, experiment, and create a practical solution that can support faster medical screening. Every stage of the project—from research and dataset preparation to model development and implementation—was carried out as part of my independent learning journey.

**Key Highlights**

* Machine Learning-based Pulmonary Edema Detection
* Medical Image Analysis using Chest X-rays
* NIH Chest X-ray Dataset
* Data Preprocessing and Feature Extraction
* Flask-based Web Application
* Real-time Prediction Interface

**Technologies Used**

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* TensorFlow
* Flask
* HTML/CSS

**Learning Outcomes**

This project helped me strengthen my understanding of Machine Learning workflows, medical imaging applications, model evaluation techniques, and healthcare-focused AI solutions. It also enhanced my ability to build end-to-end ML applications that combine both backend intelligence and user-friendly interfaces.
